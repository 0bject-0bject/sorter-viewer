# Sorter Viewer

## Description
A sorting algorithm visualizer made with rust.

## Download
1. Clone the repository
2. Run `cargo build --release`
3. The executable will be in `target/release/sorter_viewer`

## Controls
| Key | Action |
| --- | --- |
| Space | Start/Stop |
| esc | Quit |

## Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Shell Sort
- Cocktail Sort
- Any other algorithm you want to add
